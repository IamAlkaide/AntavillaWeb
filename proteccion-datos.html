2. Registro de actividades de tratamiento

3. Medidas de seguridad aplicadas

  • Las contraseñas de los socios se almacenan exclusivamente como hash SHA-256; nunca en texto plano.
  • El acceso a la zona privada requiere autenticación previa mediante email y contraseña.
  • La base de datos (Google Firestore) está protegida mediante reglas de seguridad que impiden el acceso no autorizado.
  • Las comunicaciones entre el navegador y los servidores se realizan sobre protocolo HTTPS.
  • Los datos de contacto del tablón de libros pueden ocultarse parcialmente a voluntad del socio mediante la opción de autorización.

✏️

Rectificación

Corregir datos inexactos o incompletos.

🗑️

Supresión

Solicitar la eliminación de datos cuando ya no sean necesarios.

🚫

Oposición

⏸️
